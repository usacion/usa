# usa
Asociación de inmigrantes recidentes en Estados Unidos.

Cupos;
ILIMITADOS, BAJO ESTRICTAS NORMAS DE CLASIFICACIÓN.

Idea Principal;
DESARROLLO DE PROYECTOS EN BENEFICIO DE TODOS LOS PARTICIPANTES.
_BENEFICIO ECONÓMICO.
_BENEFICIO VIVIENDA.

Quienes pueden participar;
MAYORES DE 18 AÑOS, QUIENES ESTEN DIAPUESTOS A ACATAR NORMAS Y CUMPLIRLAS.

Restricciones;
NO SE PERMITE ACTOS POLÍTICOS DE NINGUNA ÍNDOLE, LOS PARTICIPANTES ESTARÁN LIMITADOS EXCLUSIVAMENTE A LA IDEA PRINCIPAL.
